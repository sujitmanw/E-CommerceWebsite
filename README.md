# E-Commerce Website 🛒

A mobile-responsive E-Commerce website built with HTML, CSS, JavaScript and REST APIs.

## 🔗 Live Demo
👉 [Click Here to View Live](https://sujitmanw.github.io/E-CommerceWebsite/)

## 💻 GitHub Repository
👉 [github.com/sujitmanw/E-CommerceWebsite](https://github.com/sujitmanw/E-CommerceWebsite)

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- jQuery & Slick Carousel
- REST API (MockAPI)
- Font Awesome Icons
- Google Fonts

## ✨ Features
- 🏠 Home page with auto-sliding image carousel
- 👕 Product listing (Clothing & Accessories)
- 📄 Product detail page with image preview
- 🛒 Add to Cart functionality (Cookie-based)
- ✅ Order placement & confirmation page
- 📱 Fully responsive (mobile-friendly)
